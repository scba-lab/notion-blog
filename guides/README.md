# Publishing Guides

Reusable guides for content publishing workflow.

## Available Guides

### `notion-publishing.md`
Step-by-step guide for publishing blog posts to your Notion database.

**Use when**: You have a finished blog post ready to publish

### `x-posting.md`
Complete guide for posting thread content to X (Twitter).

**Use when**: You want to promote your blog post on X

### `image-design.md`
Design specifications and templates for creating social media images.

**Use when**: Creating promotional images for blog posts

## Publishing Workflow

```
Write Post → Publish to Notion → Create Social Content → Post & Track
     ↓              ↓                    ↓                    ↓
   Draft      notion-publishing    x-posting/image      Content Tracker
                                      design
```
