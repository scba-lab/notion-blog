3. Use the design specs below

### Option 2: Figma (Most Control)

## Design Specification

### Dimensions
- **Optimal Size**: 1200 x 630 px (works across all platforms)
- **Alternative**: 1080 x 1080 px (square, better for Instagram/Threads)

### Layout Design

### Color Palette (From Your Blog)
- **Background**: `#faf8f5` (cream/paper)
- **Secondary**: `#f5f2ed` (lighter cream)
- **Text**: `#1a1a1a` (ink black)
- **Accent**: `#c45a3b` (warm terracotta)
- **Border**: `#e8e4de` (subtle border)

### Typography
- **Main Title**:
  - Font: System font stack or "Inter" / "SF Pro Display"
  - Size: 64px
  - Weight: 700 (Bold)
  - Color: #1a1a1a

- **Subtitle**:
  - Font: Same as title
  - Size: 32px
  - Weight: 400 (Regular)
  - Color: #6b6b6b

- **Tech Stack**:
  - Font: Monospace ("SF Mono" / "JetBrains Mono" / "Consolas")
  - Size: 20px
  - Weight: 500
  - Color: #c45a3b

- **Footer**:
  - Font: System
  - Size: 16px
  - Weight: 400
  - Color: #6b6b6b

### Visual Elements

1. **Minimal Code Snippet** (top right corner):
```typescript
export const blog = {
  built: "3 hours",
  cost: "$0",
  tech: "Next.js + Notion"
}
```

2. **Icon/Graphic** (center or left):
- Notion logo + Next.js logo combined
- Or: Terminal window with cursor
- Or: Brackets `{ }` with gradient

3. **Subtle Grid Pattern** (background):
- Light dotted grid (#e8e4de, 20% opacity)
- Or: Notion-style checkered pattern

## Canva Template Instructions

### Step-by-Step in Canva:

1. **Create new design**: 1200 x 630 px

2. **Background**:
   - Add rectangle, fill entire canvas
   - Color: #faf8f5
   - Add subtle gradient overlay (optional)

3. **Main Title**:
   - Add text box
   - Text: "Building a Blog in One Day with AI"
   - Font: Inter Bold or Montserrat Bold
   - Size: 64px
   - Color: #1a1a1a
   - Position: Center, slightly above middle

4. **Subtitle**:
   - Add text box below title
   - Text: "Next.js 14 • Notion API • TypeScript"
   - Font: Inter Regular
   - Size: 24px
   - Color: #6b6b6b

5. **Accent Line**:
   - Add line element
   - Color: #c45a3b
   - Width: 200px, height: 3px
   - Position: Between title and subtitle

6. **Stats Box**:
   - Add text
   - Text: "3 hours build time • $0 cost • 2,369 words"
   - Font: Mono or Courier
   - Size: 18px
   - Color: #c45a3b

7. **Icon/Visual**:
   - Search Canva elements for "code brackets" or "terminal"
   - Or upload Notion logo
   - Place in corner or center

8. **Footer**:
   - Add text
   - Text: "@YourHandle • github.com/scba-lab"
   - Font: Inter Regular
   - Size: 16px
   - Color: #6b6b6b
   - Position: Bottom right

9. **Export**:
   - Download as PNG
   - Quality: High
   - Transparent background: No
